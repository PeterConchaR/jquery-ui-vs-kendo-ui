jQuery UI vs Kendo UI

We've spent a great deal of time talking to customers about Kendo UI, our framework for building modern web applications with HTML5, CSS3, and jQuery. Since its launch in November 2011, we have fielded many questions on our forums, Stack Overflow, Twitter and other networks. One of the most frequently asked questions we receive involves comparing Kendo UI with jQuery UI. On this site, I will highlight the similarities and differences between these two frameworks in the hope of answering this question in a more detailed manner than what you may have read, seen, or heard elsewhere.

Disclaimer

I work for Telerik as a Developer Evangelist. My role is to help developers be awesome with Kendo UI and other products from Telerik. That stated, my view of the world may be different than yours. My goal to present a detailed and accurate comparison of these frameworks so that you can make an informed decision if you in midst of an evaluation. By the way, if you see something that I've stuffed up, please let me know!

Introduction

Both Kendo UI and jQuery UI are built on top of jQuery.

jQuery UI

Demos & Documentation »

jQuery UI is part of the jQuery Project and provides abstractions for interactions, animations, visual effects and themeable widgets.

Kendo UI

Demos & Documentation »

Kendo UI is developed by Telerik and is comprised of three products: Kendo UI Web, Kendo UI DataViz, and Kendo UI Mobile.

Kendo UI Web
Core Kendo UI framework functionality, including the DataSource component, Templates, and drag-and-drop, plus UI widgets like Grid, ComboBox, and TreeView
Kendo UI DataViz
Collection of HTML5-powered, mobile-ready, plug-in free data visualization widgets
Kendo UI Mobile
Native UI widgets for building HTML5-powered apps and sites for modern mobile devices

Widgets

jQuery UI and Kendo UI provide widgets for constructing the front-end interface of your web application. In both frameworks, these widgets support the ability to be themed.

Important While there are fewer widgets in the list for jQuery UI, it's important to highlight that some widgets like Autocomplete combine the functionality of multiple widgets in Kendo UI Web.

Notice Menu, Menubar, Popup, Tooltip, and Spinner widgets are planned for jQuery UI 1.9. These widgets will fill in some of the gaps in the table (above). At the time of this writing, the jQuery UI team is also working on a Grid. I would recommend reviewing the jQuery UI Grid Project wiki page for more information on its status. There are plugins for other widgets available from the jQuery community.

Browser Compatibility

The widgets of both jQuery UI and Kendo UI support older browsers, including Internet Explorer 7+, Firefox 3+, Chrome, Safari 4+, and Opera 10+. jQuery UI goes a little further by supporting Internet Explorer 6. However, after receiving a great deal of feedback on the issue of supporting Internet Explorer 6 from customers, we made the decision to drop its support after Beta 2.
